# 🌿 FloraVision

A premium and responsive plant shopping landing page built with **React, Vite, Tailwind CSS, and Framer Motion**.  
FloraVision focuses on delivering a clean nature-inspired user experience with modern UI patterns, smooth animations, glass effects, and reusable component architecture.

---

## 🚀 Live Features

* 🌱 Modern plant store landing page
* 📱 Fully responsive design
* ✨ Smooth animations using Framer Motion
* 🎨 Glassmorphism UI
* 🛒 Product showcase section
* 🌿 Trending plants section
* ⭐ Customer review section
* 🍃 Oxygen plants slider
* 📧 Newsletter subscription footer
* ♻️ Reusable React components

---

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/your-username/FloraVision.git
```

Navigate to the project folder:

```bash
cd FloraVision
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Open your browser and visit:

```text
http://localhost:5173
```
